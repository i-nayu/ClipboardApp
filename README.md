# ClipboardApp 📋
macOS向けのシンプルで軽量なクリップボード履歴管理アプリです。
バックグラウンドで常にコピー内容を監視し、ショートカットキーで素早く履歴を呼び出すことができます。

## 🚀 主な機能
自動監視: 0.5秒間隔でクリップボードを監視し、テキストのコピーを自動記録。

履歴保持: 最大30件までのテキスト履歴を保存（重複は自動で最新に並び替え）。

クイック呼び出し: KeyboardShortcuts を使用したグローバルショートカット対応。（ショートカットキーは自由に設定可能）

スマートUI: 履歴から項目を選択すると、自動でシステムクリップボードにセットし、アプリを隠します。

## 🛠 技術スタック
Swift / SwiftUI

## ⌨️ 使い方
アプリを起動すると、バックグラウンドで監視が始まります。

テキストをコピーすると、自動的に履歴（history）に追加されます。

設定したショートカットキー（toggleClipboard）を押すと、マウスカーソルの位置に履歴ウィンドウが表示されます。

リスト内の項目をクリックすると、その内容がクリップボードにセットされ、ウィンドウが閉じます。

## ⚙️ 設定
他のアプリのコピー内容を読み取るため、特別な設定は不要ですが、自動ペースト機能を拡張する場合は「アクセシビリティ」の許可が必要になる場合があります。

pc起動時に同時にアプリを開くようにすると便利です
